# feng3d

> feng3d是使用TypeScript进行编写基于WebGL的3D游戏引擎，致力于打造一个优秀的3d游戏引擎以及易用且可以快速开发项目的配套编辑器。

## 特性

1. 提供优质的typescript编辑器
1. 支持脚本，你能够在这里实现所有你想做的事情。
1. 支持自定义材质，只需要你提供shader代码以及所有到的数据结构就可以实现你想要的任何渲染效果（目前高级效果除外）。
